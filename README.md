# Stress Level Detection System

## Project Description

The **Stress Level Detection System** is a machine learning based application that analyzes speech and text inputs to detect stress levels.
The system processes emotional patterns from audio signals and textual data to classify stress into different categories such as **Low, Moderate, and High stress**.

This project aims to assist in early detection of stress and promote better mental well-being.

## Machine Learning Techniques

The system uses machine learning models to classify stress levels from extracted features.

Models used:

* Random Forest
* Support Vector Machine (SVM)

Audio features such as **MFCC (Mel Frequency Cepstral Coefficients)** are extracted for emotion recognition.

---

## Datasets Used

The project uses publicly available emotional speech datasets:

1. RAVDESS – Ryerson Audio-Visual Database of Emotional Speech and Song
   https://zenodo.org/record/1188976

2. TESS – Toronto Emotional Speech Set
   https://tspace.library.utoronto.ca/handle/1807/24487

3. EmoDB – Berlin Emotional Speech Database
   http://emodb.bilderbar.info/start.html

```

Install frontend dependencies.

```
